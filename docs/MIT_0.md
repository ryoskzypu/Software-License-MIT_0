# NAME

Software::License::MIT\_0 - MIT No Attribution License (MIT-0)

# SYNOPSIS

```perl
use Software::License::MIT_0;

my $license = Software::License::MIT_0->new(
    {
        holder => 'COPYRIGHT HOLDER',
    }
);

print $license->license;
```

# DESCRIPTION

This module is a [Software::License](https://metacpan.org/pod/Software%3A%3ALicense) subclass implementation of the MIT No Attribution
License. The "MIT No Attribution" or "MIT-0" license is a variant of the permissive
MIT license that removes the requirement for attribution.

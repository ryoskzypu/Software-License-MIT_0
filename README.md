# Software::License::MIT_0

This module is a [Software::License](https://metacpan.org/pod/Software::License)
subclass implementation of the MIT No Attribution License. The "MIT No Attribution"
or "MIT-0" license is a variant of the permissive MIT license that removes the
requirement for attribution.

## Installation

To download and install this module directly with [cpanminus](https://metacpan.org/pod/App::cpanminus):

```shell
$ cpanm https://github.com/ryoskzypu/Software-License-MIT_0.git
```

To do it manually, run the following commands (after cloning the repository):

```shell
$ cd Software-License-MIT_0
$ perl Makefile.PL
$ make
$ make test
$ make install
```
